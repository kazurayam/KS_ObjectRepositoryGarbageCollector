## Building jar in the `lib` project

[source,text]
```
$ cd $rootProject/lib
$ gradle jar
```

## Importing jars from the `lib` sub-project into the `katalon` sub-project

[source,text]
```
$ cd $rootProject/katalon
$ gradle importDrivers
```
