[Groovy] Class ObjectRepositoryDecorator

    • com.kazurayam.ks.testobject.ObjectRepositoryDecorator 

    • class ObjectRepositoryDecorator
extends java.lang.Object

      ObjectRepositoryDecorator implements methods that returns a list of TestObjectId, a list of TestObjectEssenses, that match with the pattern for Locators. ObjectRepositoryDecorator enables to look up Test Objects by Locators.

        • LocatorIndex getLocatorIndex(java.lang.String pattern, java.lang.Boolean isRegex)

          LocatorIndex is a list of "Locators", each of which associated with the list of TestObjectEssence objects which have the same "Locator" string. You should pay attention to the locators that has 2 or more belonging TestObjectEssence objects; as it means you have duplicating TestObjects with the same Locator.
